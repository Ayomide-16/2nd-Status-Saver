## Installation

### From Releases
Download the latest APK from the [Releases](https://github.com/Ayomide-16/2nd-Status-Saver/releases) page.

### Build from Source
```bash
git clone https://github.com/Ayomide-16/2nd-Status-Saver.git
cd 2nd-Status-Saver
flutter pub get
flutter build apk --release
```

## Usage

1. Launch the app
2. Grant storage permission when prompted
3. Select the WhatsApp status folder: `Android/media/com.whatsapp/WhatsApp/Media/.Statuses`
4. View statuses in the Live tab
5. Tap to view full-screen, tap save button to save

## Requirements

- Android 5.0 (API 21) or higher
- WhatsApp installed
